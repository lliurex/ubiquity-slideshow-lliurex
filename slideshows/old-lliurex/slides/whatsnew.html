What's new in LliureX 12.06?

These are some of the new features in LliureX Nemo 12.06 release:

  • New release calendar: in June instead of September
  • Kernel updated to 2.6.38 to support new hardware
  • Easier Classroom model installation
  • Changed from OpenOffice 3.2 to LibreOffice 3.4.5
  • Several software updates like: Firefox 12, Thunderbird 12, Scribus 4, GCompris 12, Tuxtype 1.8, TuxPaint 0.9, Geogebra 4.0, JClic resource collections, etc.
  • New in-house developments: Flash Video Saver, LliureX Homework Harvester, LliureX Freeze, etc.
