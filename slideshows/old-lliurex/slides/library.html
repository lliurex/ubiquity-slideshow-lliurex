Library Adaptation

  • LliureX Library is an adaptation designed for libraries at schools, which includes the application PMB for the management and control of libraries.
  • It comprises tools for bulk data input from GesCen, Itaca, external libraries (through Z39.50 standard) and other databases via CSV format. This way the library manager can introduce a huge amount of book information with just a few clicks.
  • It also has the expected features as spine label printing, book reservations, reader management, etc.
